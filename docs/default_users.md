# Default Users

This document lists the default users created during the initial database setup and migration. 

## Admin User
- **Email:** `admin@helphub.lk`
- **Password:** `admin` 
- **Role:** `ADMIN`
- **Description:** System administrator account with full access to all roles, permissions, and profiles.

*(Note: It is highly recommended to change this password or disable this default account after your initial setup in production environments.)*
